# Brumble

Landing page and brand assets for **Brumble**, a tiny bee strategy game.

Served with GitHub Pages from the repo root.

## Assets

| File | Use |
| --- | --- |
| `assets/logo.svg` | Master logo (scales to any size, needs a browser for the web font) |
| `assets/logo-2000x1200.png` | Logo render, white background |
| `assets/logo-transparent-2000x1200.png` | Logo render, transparent background |
| `assets/brumble-avatar-1024.png` | Square avatar, source size |
| `assets/brumble-avatar-400.png` | X / TikTok profile picture, optimized |
| `assets/brumble-banner-1500x500.png` | X header banner |

## Regenerating PNGs

```
python -m http.server 8741
python render_social.py
```
